Deploy to GitHub

Host ClassPulse independently for free

Why GitHub Pages?

GitHub Pages lets you host this tool as a permanent, shareable URL — completely free. Your data stays local in the browser.

  • 1
    Create a GitHub account at github.com if you don't have one.
  • 2
    Create a new repository. Click the + icon → "New repository". Name it classpulse. Set it to Public.
  • 3
    Download this file from your browser (File → Save Page As) and save it as index.html
  • 4
    Upload the file. In your repository, click "Add file" → "Upload files" → select your index.html.
  • 5
    Enable GitHub Pages. Go to your repo → Settings → Pages → Source: "Deploy from a branch" → Branch: main → Save.
  • 6
    Your site goes live at:
    https://YOUR-USERNAME.github.io/classpulse/
    Replace YOUR-USERNAME with your GitHub username.

📦 Backup & Restore Your Data

Your data is stored in the browser. Export it before switching devices or clearing browser storage.
