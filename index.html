Deploy & Install

Use ClassPulse as a native-feeling iOS app

Add to iPhone Home Screen

ClassPulse works as a Progressive Web App (PWA). Once hosted, you can install it on your iPhone and it will look and behave exactly like a native app — full screen, no browser bar.

1
Host this file on GitHub Pages (free — see below)
2
Open the URL in Safari on your iPhone
3
Tap the Share button
4
Tap "Add to Home Screen" → Done! 🎉

GitHub Pages — Free Hosting in 5 Minutes

Host ClassPulse at a permanent URL for free. Your data stays private in the browser.

  • 1
    Create a GitHub account at github.com if you don't have one.
  • 2
    New repository → name it classpulse → set to Public.
  • 3
    Save this page (File → Save Page As) as index.html
  • 4
    Upload → "Add file" → "Upload files" → select index.html → Commit.
  • 5
    Enable Pages: Settings → Pages → Branch: main → Save.
  • 6
    Your app goes live at:
    https://YOUR-USERNAME.github.io/classpulse/

📦 Backup & Restore Data

Data lives in your browser. Export before switching devices.
